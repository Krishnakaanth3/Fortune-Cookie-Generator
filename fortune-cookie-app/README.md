# 🥠 Fortune Cookie Generator

A beautiful, interactive fortune cookie app with dark/light themes, 360° rotatable cookies, smooth animations, and social sharing features.

## ✨ Features

- **🎯 Interactive Fortune Cookie**: 360° rotatable cookie in the center of the screen
- **📜 Paper Pull Animation**: Smooth animation to pull out the fortune paper from bottom to top
- **🌙 Dark/Light Mode**: Pure luxury white and matte black themes
- **💫 1000 Unique Fortunes**: Emotionally valuable messages that bring smiles
- **⏰ Timer System**: 30-minute cooldown with countdown to next cookie (23:59:59)
- **📱 Social Sharing**: Share fortunes on social media (appears 10 seconds after reveal)
- **🎨 Beautiful Shadows**: Appropriate shadows for both light and dark modes
- **📱 Responsive Design**: Works perfectly on all devices

## 🚀 Quick Start

### Local Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/Fortune-Cookie-Generator.git
   cd Fortune-Cookie-Generator/fortune-cookie-app
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   - Navigate to `http://localhost:5173`

### Production Build

```bash
npm run build
npm run preview
```

## 🌐 Deployment

### GitHub Pages (Automated)

1. **Fork this repository**
2. **Enable GitHub Pages** in repository settings
3. **Update the homepage** in `package.json`:
   ```json
   "homepage": "https://your-username.github.io/Fortune-Cookie-Generator"
   ```
4. **Push to main branch** - automatic deployment via GitHub Actions

### Manual GitHub Pages Deployment

```bash
npm run deploy
```

### Other Hosting Platforms

- **Vercel**: Connect your GitHub repo to Vercel
- **Netlify**: Drag and drop the `dist` folder after `npm run build`
- **Firebase**: Use Firebase CLI to deploy the `dist` folder

## 🎮 How to Use

1. **Click the fortune cookie** to open it (only available once per session)
2. **Grab and pull the paper** from bottom to top to reveal your fortune
3. **Read your personalized message** designed to bring a smile
4. **Share on social media** when the sharing modal appears (after 10 seconds)
5. **Wait for the timer** - next cookie available after the countdown
